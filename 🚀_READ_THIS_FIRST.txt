🚀 GET STARTED IN 3 STEPS (2 MINUTES)
───────────────────────────────────────────────────────────────

Step 1: Install Dependencies
────────────────────────────
Open terminal in this folder and run:

    npm install

Step 2: Add OpenAI API Key
───────────────────────────
1. Get API key from: https://platform.openai.com/api-keys
2. Create file named: .env.local
3. Add this content:

    OPENAI_API_KEY=sk-your-actual-key-here
    SITE_URL=http://localhost:3000

Step 3: Start Development Server
─────────────────────────────────
Run:

    npm run dev

Then open: http://localhost:3000

💰 MONETIZE (OPTIONAL)

Add Google AdSense:

2. Get approved (1-2 weeks)
3. Add codes to .env.local:
   NEXT_PUBLIC_ADSENSE_CLIENT=ca-pub-xxxxxxxxxxxx
   NEXT_PUBLIC_ADSENSE_SLOT_TOP=xxxxxxxxx
   NEXT_PUBLIC_ADSENSE_SLOT_MID=xxxxxxxxx
   NEXT_PUBLIC_ADSENSE_SLOT_BOTTOM=xxxxxxxxx
4. Restart server

Ads appear automatically!
